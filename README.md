# Skills

A repository of `SKILL.md` files for AI agents and automation tools.

## Structure

Each subdirectory contains a `SKILL.md` file that documents a specific technology or domain. Agents can read these files to understand key concepts, common tasks, and best practices for each skill area.

| Skill | Description |
|-------|-------------|
| [git](git/SKILL.md) | Git version control workflows and best practices |
| [linux](linux/SKILL.md) | Linux system administration and command-line usage |
| [python](python/SKILL.md) | Python development patterns and tooling |
| [docker](docker/SKILL.md) | Docker container build and management |
| [kubernetes](kubernetes/SKILL.md) | Kubernetes deployment and operations |

## Adding a New Skill

1. Create a new subdirectory: `mkdir <skill-name>`
2. Add a `SKILL.md` file following the existing structure:
   - **Overview** – What this skill covers
   - **Key Concepts** – Core ideas and terminology
   - **Common Tasks** – Step-by-step instructions with code examples
   - **Best Practices** – Recommended approaches
   - **References** – Further reading and documentation links
3. Update the table above in this README and in the root [SKILL.md](SKILL.md).
